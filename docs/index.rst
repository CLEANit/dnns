Deep Neural Networks
===============================================

This module provides scripts, classes, and functions to help you train deep neural networks on multiple machines with multiple GPUs. This module utilizes the following libraries (in no particular order):

    - h5py
    - pytorch
    - nvidia-apex
    - yaml 
    - cmocean
    - numpy
    - matplotlib

To get started, :doc:`click here <./gettingstarted>`.
